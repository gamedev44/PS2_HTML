PS2 -Online Web Sound Test

Controls:

Press Left or Right Arrow Keys to hear the Option Selected Sound.

Press Enter Key to hear the Game Disc Select Sound.

Purpose of this page:

This page allows you to test and debug the sound playback when specific keys are pressed. It plays different sounds based on the Enter or Arrow keys, which is useful for debugging web-based emulators or sound effects.

Use the Enter key for the Game Disc Start sound and the Arrow keys for navigation sound.
